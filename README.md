## Objetivo:
O objetivo deste projeto é praticar criação de API REST e utilizar como consulta de estudo.
Através dos métodos: GET, POST, PUT, DELETE os dados serão consumidos no banco em memória (H2 SQL). 
Simula CRM de cliente, com apenas os atributos id e nome.

## Aprendizados
- Iniciar projeto com spring boot
- Algumas das Annotations do spring, JPA e Lombok
- Banco de dados em memória (H2 SQL)
- Objetivo da Interface nesse contexto (classe responsável por interagir com o banco de dados para uma determinada entidade).


## **Como Usar**
- Ter o JDK instalado
- Para compilar e executar: através de uma IDE que possa compilar e executar Java (ex: IntelliJ, Eclipse)
- Alguma ferramenta para testar API REST (exemplo: Postman).
